# Online Examination System

A modern desktop-based Online Examination System developed using Python, Tkinter, and SQLite with a professional TCS-style interface.

---

## Overview

This project is designed to conduct online examinations in a secure and user-friendly desktop environment.  
It includes student and admin modules, real-time exam monitoring, performance analytics, and detailed result analysis.

---

## Features

- Secure Student & Admin Login
- TCS-Style Modern User Interface
- Real-Time Countdown Timer
- Multiple Choice Questions (MCQs)
- Negative Marking System
- Mark for Review Functionality
- Question Navigation Panel
- Instant Result Generation
- Detailed Question Analysis
- Performance Analytics Dashboard
- Leaderboard System
- SQLite Database Integration
- Admin Question Management System

---

## Technologies Used

- Python
- Tkinter
- SQLite3
- Pillow (PIL)

## Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Online-Examination-System.git
```

### 2. Navigate to Project Folder

```bash
cd Online-Examination-System
```

### 3. Install Required Package

```bash
pip install pillow
```

### 4. Run the Application

```bash
python online_exam_system.py
```
